# Live Nation Entertainment (live-nation-entertainment)
Live Nation Entertainment is the world's leading live entertainment company, operating Ticketmaster, the global event ticketing platform. Through the Ticketmaster Developer Portal, the company provides the Discovery API enabling developers to search for events, attractions, venues, and classifications across all supported sources, markets, and locales with access to content from Ticketmaster, Universe, FrontGate Tickets, and Ticketmaster Resale.

## APIs

### Ticketmaster Discovery API
The Ticketmaster Discovery API allows developers to search for events, attractions, venues, and classifications across all Ticketmaster supported sources, markets, and locales. The API provides access to content from Ticketmaster, Universe, FrontGate Tickets, and Ticketmaster Resale platforms with a default quota of 5000 API calls per day and rate limitation of 5 requests per second.
